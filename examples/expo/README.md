# Expo Example

Placeholder for an Expo sample app that depends on `packages/js` (`@swift-man/material-design-color`).

Use `createMaterialTheme({ preset: "expressive" })` and `theme.colorScheme` for Material 3 preset role colors. Use `createMaterialTheme({ colorScheme: { primary: "#6750A4" } })` to inject Material Theme Builder role colors.
